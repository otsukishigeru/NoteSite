# プロジェクト概要
このプロジェクトは「思考の技法」の発信サイトです。
著者：大槻 繁（おおつき しげる）
シリーズ名：思考の技法 新訂版

# Claude Code への指示
文章を書くときは必ず skills/TMBook_Skill.md を参照すること。

# 文章生成の基本ルール
- 敬体（です・ます調）で統一する
- 重要な概念は〈 〉で囲む
- 各セクション末尾に ■ を置く
- 過度な箇条書きは避け、段落形式で書く

# フォルダ構成
- skills/TMBook_Skill.md … 文体・思想の指示書
- index.html … トップページ
- （記事ファイルはルート直下に置く）

# やってはいけないこと
- 箇条書きで本文を書かない（段落形式を守る）
- です・ます調を崩さない
- 〈 〉なしで重要概念を使わない
- 著者の一人称を「私」にしない（「筆者」を使う）

# よく使う作業
- 新しい記事を追加するときは skills/TMBook_Skill.md を参照して生成する
- 記事完成後は git add . → git commit → git push の順で保存する

# 更新運用ルール
## note新作追加時（随時）
- note に新作を公開したら index.html のナビゲーションに追記する
- 追記時は記事タイトル・カテゴリ・URLを必ず伝える

## 定期見直し（3か月ごと）
- 主要メッセージ20件を組み替える
- 現行の見栄えを踏襲しつつ、項目のみ入れ替える
- 見直し後は git add . → git commit → git push で保存する
